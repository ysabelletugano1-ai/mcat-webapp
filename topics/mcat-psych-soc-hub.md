## Tutor Strategy

1. **Learn the theory names AND their authors** — MCAT Psych/Soc loves attribution questions: "Which theory predicts X?" Know Durkheim, Marx, Weber, Mead, Piaget, Erikson, Kohlberg, Bandura, Milgram, Asch by name.
2. **Sociology = macro; Psychology = micro** — When unsure whether a concept is psych or soc, macro-level (society, institutions, groups) = soc; micro-level (individual, cognition, behavior) = psych.
3. **Biological bases are tested** — Know the neurobiology behind memory (hippocampus), fear (amygdala), reward (nucleus accumbens/dopamine), motor (cerebellum, basal ganglia). Psych/Soc has ~5% biology.
4. **Every experiment question has a variable structure** — Independent variable = what the researcher manipulates; dependent variable = what is measured. Social loafing, bystander effect, Stanford Prison Experiment — identify IV/DV for each.
5. **Health disparity questions anchor on SES and power** — If a passage describes a health gap between groups, the MCAT answer usually involves access, power, or structural barriers — not individual choices.

## Common Traps

- **Positive punishment vs. negative reinforcement** — Positive/negative = add/remove a stimulus. Reinforcement/punishment = behavior increases/decreases. Negative reinforcement (removing something bad) INCREASES behavior.
- **Encoding vs. retrieval failure** — Information can be stored but not retrieved (tip-of-the-tongue) vs. never encoded. Context-dependent memory involves retrieval, not encoding.
- **Prejudice vs. discrimination** — Prejudice = attitude (internal). Discrimination = behavior (action). You can have one without the other.
- **Organic vs. functional disorders** — Organic = identifiable biological cause. Functional = no clear biological cause. MCAT distinguishes these in case passages.
- **Intergenerational vs. intragenerational mobility** — Inter = across generations (parent → child). Intra = within a single lifetime.
