## ⚡ Shortcuts Index

*All tutor-verified expert shortcuts across all topic cards. Sweep this list during final review week.*

### Bio/Biochem
- **Enzyme Kinetics** — Lineweaver-Burk intersection tells you the inhibition type: Y-axis → competitive (Vmax same); X-axis → noncompetitive (Km same); parallel lines → uncompetitive (both change). → [[cards/bio-biochem/enzyme-kinetics]]
- **Protein Structure** — Denaturation breaks secondary/tertiary/quaternary (H-bonds, disulfide, hydrophobic). Primary structure (peptide bonds) survives denaturation. → [[cards/bio-biochem/protein-structure]]
- **DNA Replication** — Leading strand = continuous, toward fork. Lagging = Okazaki fragments, away from fork. Helicase → primase → Pol III → Pol I (removes primer) → ligase. → [[cards/bio-biochem/dna-replication]]
- **Transcription/Translation** — Start: AUG. Stop: UAA/UAG/UGA ("U Are Gone, U Go Away, U Are Away"). Template strand read 3'→5'; mRNA synthesized 5'→3'. → [[cards/bio-biochem/transcription-translation]]
- **Glycolysis/Respiration** — Net ~36 ATP/glucose: 2 glycolysis + 2 Krebs + ~32 ETC. Krebs runs twice per glucose. PFK-1 = rate-limiting step of glycolysis. → [[cards/bio-biochem/glycolysis-respiration]]
- **Cell Membrane** — Na⁺/K⁺ ATPase: 3 Na⁺ out, 2 K⁺ in. Net negative charge inside → maintains resting potential. → [[cards/bio-biochem/cell-membrane-transport]]
- **Cell Cycle** — Cyclins rise and fall; CDKs are constant. G1 checkpoint: Rb/p53. Lost checkpoints = cancer. → [[cards/bio-biochem/cell-cycle-mitosis]]
- **Nervous System** — Depolarization: Na⁺ IN. Repolarization: K⁺ OUT. Hyperpolarization: K⁺ channels slow to close. Absolute refractory: no AP possible. → [[cards/bio-biochem/nervous-system]]
- **Cardiovascular** — CO = HR × SV. Frank-Starling: more preload → stronger contraction (up to a limit). → [[cards/bio-biochem/cardiovascular-system]]
- **Endocrine** — Steroid hormones: lipid-soluble, cross membrane, genomic effects (slow). Peptide hormones: water-soluble, surface receptor, second messenger (fast). → [[cards/bio-biochem/endocrine-system]]
- **Immune** — MHC I: all nucleated cells → CD8+ (cytotoxic). MHC II: APCs only → CD4+ (helper). "1×1=1, 2×2=4" — MHC I with CD8, MHC II with CD4. → [[cards/bio-biochem/immune-system]]

### Chem/Physics
- **Acid-Base** — pH = pKa + log([A⁻]/[HA]). When pH = pKa: 50/50 ratio. Above pKa: more base. Below: more acid. → [[cards/chem-physics/acid-base-chemistry]]
- **Thermodynamics** — ΔG = ΔH − TΔS. Negative ΔH + positive ΔS = always spontaneous. Positive ΔH + negative ΔS = never spontaneous. → [[cards/chem-physics/thermodynamics]]
- **Electrochemistry** — E°cell = E°cathode − E°anode. Positive = spontaneous. LEO GER: Loss Electrons = Oxidation, Gain Electrons = Reduction. → [[cards/chem-physics/electrochemistry]]
- **Fluid Dynamics** — Bernoulli: fast flow = low pressure. Poiseuille: flow ∝ r⁴. Doubling radius → 16× more flow. → [[cards/chem-physics/fluid-dynamics]]
- **Optics** — Negative image distance = virtual image (same side as object, upright). Positive = real (opposite side, inverted). → [[cards/chem-physics/optics]]
- **Kinetics/Equilibrium** — Catalyst speeds up equilibrium, does NOT shift it. Temperature increase shifts toward endothermic side. → [[cards/chem-physics/kinetics-equilibrium]]
- **Amino Acids** — Only Asp/Glu (acidic) and Lys/Arg/His (basic) have charged R groups at pH 7.4. His changes charge near physiological pH — key for enzyme active sites. → [[cards/chem-physics/amino-acids-organic]]

### Psych/Soc
- **Classical Conditioning** — CS predicts UCS. Extinction: CS alone → CR fades. Spontaneous recovery: CR returns after rest without reconditioning. → [[cards/psych-soc/classical-operant-conditioning]]
- **Memory** — Hippocampus = explicit memory. Amygdala = emotional memory. Cerebellum = procedural. Short-term: 7±2 items, ~30 sec. → [[cards/psych-soc/memory-systems]]
- **Developmental Stages** — Piaget: "Silly People Can't Function" = Sensorimotor, Preoperational, Concrete, Formal. → [[cards/psych-soc/developmental-stages]]
- **Social Influence** — More bystanders = less helping (diffusion of responsibility). Asch = conformity; Milgram = obedience to authority. → [[cards/psych-soc/social-influence]]
- **Sociological Theories** — Durkheim = Harmony (functionalism); Marx = War (conflict); Mead = Meaning (symbolic interactionism). → [[cards/psych-soc/sociological-theories]]
- **Health Disparities** — SES is the strongest single predictor of health outcomes. Every health disparity question frames around access, power, or resources. → [[cards/psych-soc/health-disparities]]
- **Psych Disorders** — Schizophrenia: positive symptoms = things added (hallucinations, delusions); negative = things taken away (flat affect). → [[cards/psych-soc/psychological-disorders]]
- **Social Stratification** — Open system = class (mobility possible). Closed = caste (mobility rare). Intergenerational = across generations; intragenerational = within one lifetime. → [[cards/psych-soc/social-stratification]]
